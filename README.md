# FlipTrack

Suivi d'achat-revente connecté à Supabase.

## Déploiement Vercel

1. Importe ce dossier sur GitHub
2. Connecte le repo sur vercel.com
3. Dans Vercel → Settings → Environment Variables, ajoute :
   - `VITE_SUPABASE_URL` = ton URL Supabase
   - `VITE_SUPABASE_ANON_KEY` = ta clé anon

4. Clique Deploy ✓
